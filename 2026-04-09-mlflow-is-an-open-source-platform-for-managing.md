# MLflow on OpenShift AI

- Date: 2026-04-09
- Summary: MLflow is an open-source platform for managing the machine learning lifecycle, including experiment tracking, model evaluation, model registry, and deployment. It is integrated into OpenShift AI as a key tool for data science and data engineering.

## Notes
MLflow is an open-source platform designed to streamline the machine learning lifecycle. It offers four primary components:
*   **MLflow Tracking:** Records and queries experiments, including code, data, configuration, and results.
*   **MLflow Projects:** Packages ML code in a reusable and reproducible format.
*   **MLflow Models:** Manages and deploys models from various ML libraries to diverse serving platforms.
*   **MLflow Model Registry:** Provides a centralized hub for managing the full lifecycle of MLflow Models, including versioning, stage transitions, and annotations.

On Red Hat OpenShift AI, MLflow serves as a crucial tool for data science and data engineering practitioners. It enables users to manage their machine learning workflows efficiently within the OpenShift environment, facilitating experiment tracking, model management, and deployment of AI models.
